# KoboToolbox Data Collection Forms

A central repository for KoboToolbox forms, supporting assets, and related work projects.

## Purpose

- Store KoboToolbox form definitions and versions
- Organize related documentation and reference materials
- Keep future work projects in one structured place

## Suggested Structure

```text
forms/          KoboToolbox forms and exports
docs/           Notes, guides, and specifications
projects/       Related implementation work
assets/         Supporting files and media
```

## Notes

- Add each new project in its own subfolder
- Keep sensitive credentials and exported private data out of version control
