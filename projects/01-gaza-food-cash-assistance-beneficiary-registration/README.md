# Gaza Food and Cash Assistance Beneficiary Registration

## Overview

This project is a KoboToolbox beneficiary registration form prepared for food and cash assistance workflows in Gaza. The form is designed to support structured field registration with a clear Arabic interface and a practical data collection flow for humanitarian use cases.

## Project Goal

The form is intended to capture beneficiary and household information in a consistent format that can be uploaded and managed through KoboToolbox for registration and screening activities.

## Form Highlights

- Humanitarian beneficiary registration workflow
- Arabic form interface
- Section-based question grouping
- Structured applicant identity fields
- Household and shelter status questions
- Enumerator information section
- Kobo-ready XLSForm structure
